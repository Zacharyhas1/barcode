Use the toggle to switch between barcode and QR code mode. Click the checkmark to save codes in your local storage. Long-press the checkmark to create a code that doesn't display the value in plain text (for e.g. passwords). Double-tap a saved code to edit it or add an alias. Drag to re-order. Use 📁 to create folders, and the ▸ button to move codes between folders. Use 📷 to scan a barcode or QR code with your camera or from an image.
